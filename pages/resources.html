Skeleton HTML Page

Use this as the starting script for your web pages. (Change the title and change the stylesheet to match your actual stylesheet location.)

<!DOCTYPE html>
<html lang="en">
   <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document Title</title>
        <link href="css/stylesheet.css" rel="stylesheet" type="text/css">
    </head>
    <body>

   </body>
</html>

Web Resources for HTML with CSS

These web resources for the topics below can be used in the syllabus in lieu of a textbook for an introductory HTML with CSS class.

  1. W3Schools is great for both quick reference, trying it yourself or tutorials, especial their HTML Tutorial.
    There's also a Russian version
  2. Shay Howe's Learn to Code HTML & CSS
  3. Coursera has an excellent 5-part series Web Design for Everybody: Basics of Web Development & Coding Specialization from University of Michigan.
  4. Udacity is an excellent free training program. Easy to use and very clear.  Here is their Html with CSS class.
  5. LinkedIn Learning - students may be able to use free with their school account: FREE LinkedIn Learning for College Students (Step by Step).
  6. Mozilla developer network - More technical than the other but guides for beginner through expert concepts.
  7. Free Code Camp - in browser learning simple to use plus certificates.
  8. Watch the Coursera video Editors: How to Use an Editor to Create an HTML File to see how VS Code can create a skeleton web page automatically for you.
  9. Life Wire's Recommended Text Editors

For your text editor, check out the recommendations on our IDE page. Replit, Visual Studio Code, and Brackets are all good choices. Notepad++ can be used, but it doesn't give you as many code editing features. Do not use MS Word or Wix or any other tool to autogenerate HTML code for you, because they put unnecessary junk in your page, and to be a skilled developer you need to learn the HTML syntax for yourself.

Cheat Sheets

HTML encoding is all about the tags. You are not expected to have them all memorized, especially in the beginning, but you will need to be fast looking them up. The common ones you will use so much you will memorize them without even trying. W3 Schools is the definitive source for tag definitions, so bookmark this: https://www.w3schools.com/tags.
